설치 안내

1. initial_setting.bat 실행

정상적인 확장 프로그램 구동을 위해 initial_setting.bat 파일을 다운로드하여 실행해주세요.
해당 파일은 Python 3.11.4와 Pip를 설치하는 파일입니다. 사용자의 컴퓨터에 악영향을 미칠 코드는 없습니다.

Download initial_setting.bat

2. 프로필 경로 확인

Chrome에서 내 프로필 경로를 확인하는 방법

  1. 오른쪽 url chrome://version/ 을 클릭하여 URL을 복사합니다.

  2. 새 탭을 열고 복사한 URL을 붙여넣은 후 Enter를 누릅니다.
  3. "프로필 경로" 항목을 찾습니다.
  4. 표시된 경로를 복사하여 아래에 입력해주세요.

3. 프로필 경로 입력

4. 확장 프로그램 설정

파일 탐색기에서 F4를 누르고 생성된 경로를 입력하여 확장 프로그램이 설치된 폴더로 이동합니다.
이동한 폴더에서 install_host.bat 파일을 실행하세요.
이 파일은 확장 프로그램이 로컬 컴퓨터와 소통하기 위한 설정을 세팅합니다.
